# AI进化论 - 博客

让AI成为你的第二大脑。每周3篇实战文章，探索AI工具与自动化工作流。

## 部署

本博客使用纯静态HTML，无需构建，直接部署到GitHub Pages即可。

### 快速部署到 GitHub Pages

1. 在GitHub创建新仓库，例如 `ai-blog`
2. 将本目录内容推送到仓库的 `gh-pages` 分支
3. 在仓库 Settings → Pages → Source 选择 `gh-pages` 分支
4. 等待1-2分钟，博客即可访问

### 目录结构

```
ai-blog/
├── index.html          # 首页
├── articles/           # 文章目录
│   ├── claude-day-template.html
│   └── chatgpt-article-prompt.html
├── sitemap.xml         # SEO地图
├── robots.txt          # 爬虫协议
└── README.md
```
